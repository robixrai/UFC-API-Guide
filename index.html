UFC Fighters API v1.1

The ultimate data source for MMA fighter stats and real-time rankings.

Base URL

https://ufc-predictor-v1.onrender.com

Fighters

GET /fighters

Returns a simple list of all fighter names available in the database. Supports optional filtering.

Optional Parameters gender , division

Example Request
https://ufc-predictor-v1.onrender.com/fighters?gender=men's&division=Lightweight

Tip: encode special characters in query strings. Example: men%27s instead of men's.

GET /fighters/{name}

Returns full details for one fighter. Supports partial name searches. Use the block parameter to filter data categories.

Optional Parameter block

  • personal-info, career, specs, description
Example Request
https://ufc-predictor-v1.onrender.com/fighters/Charles%20Oliveira?block=career

Rankings

GET /rankings/champions

Returns a nested dictionary of every champion in every division.

Optional Parameters gender, division

https://ufc-predictor-v1.onrender.com/rankings/champions?gender=men's&division=Lightweight

Tip: you can request just the champions for a gender (?gender=men's) or a single division (?division=Lightweight).

GET /rankings/divisions

Returns a list of all division names.

Optional Parameters gender

https://ufc-predictor-v1.onrender.com/rankings/divisions?gender=men's
GET /rankings/{gender}/{division}

Returns rankings for a specific weight class. If a rank query parameter is provided, the endpoint returns only the fighter at that position.

Optional Parameter rank

Flexible path: you may call /rankings/{gender} to get rankings for all divisions of that gender; /rankings/{gender}/{division} returns only that division. Optional parameter rank returns the fighter who is that rank in that specific division.

https://ufc-predictor-v1.onrender.com/rankings/men's/lightweight?rank=1

Examples: /rankings/men's → all men's divisions; /rankings/men's/lightweight → lightweight rankings only.

GET /rankings/search/{name}

Returns the rank(s) of a fighter, including their division and gender. Returns all matches for the provided name fragment.

https://ufc-predictor-v1.onrender.com/rankings/search/Charles%20Oliveira

Python Example

import requests

BASE = "https://ufc-predictor-v1.onrender.com"

# Example: Get stats for Charles Oliveira
fighter_url = f"{BASE}/fighters/Charles Oliveira"

try:
    response = requests.get(fighter_url, timeout=60)
    if response.status_code == 200:
        data = response.json()
        print(data)
except Exception as e:
    print(f"Connection Error: {e}")
